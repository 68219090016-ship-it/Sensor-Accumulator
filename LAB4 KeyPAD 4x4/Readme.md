## 4x4 Keypad
![4x4 Keypad](https://abacasstorageaccnt.blob.core.windows.net/cirkit/8011ee4c-40a1-48e2-9605-1aaf9c470a5c.png "4x4 Keypad")
> แป้นพิมพ์ 4x4 คือแป้นพิมพ์แบบเมทริกซ์ที่ประกอบด้วยปุ่ม 16 ปุ่ม เรียงกันเป็น 4 แถวและ 4 คอลัมน์ นิยมใช้สำหรับการป้อนข้อมูลของผู้ใช้ในอุปกรณ์อิเล็กทรอนิกส์ ช่วยให้ผู้ใช้สามารถป้อนข้อมูลตัวเลขหรือตัวอักษรและตัวเลขได้ และมักใช้ในแอปพลิเคชันต่างๆ เช่น ระบบรักษาความปลอดภัย เครื่องคิดเลข และระบบฝังตัว การออกแบบที่กะทัดรัดและอินเทอร์เฟซที่ใช้งานง่ายทำให้เป็นตัวเลือกยอดนิยมสำหรับโครงการที่ต้องการการโต้ตอบกับผู้ใช้

# ข้อกำหนดทางเทคนิค
> แป้นพิมพ์ 4x4 ทำงานเหมือนเมทริกซ์ของสวิตช์ โดยแต่ละปุ่มจะเชื่อมต่อกับแถวและคอลัมน์ที่เฉพาะเจาะจง การกดปุ่มจะทำให้วงจรระหว่างแถวและคอลัมน์ที่เกี่ยวข้องสมบูรณ์ ทำให้ไมโครคอนโทรลเลอร์สามารถตรวจจับการกดปุ่มได้

# รายละเอียดทางเทคนิคที่สำคัญ
จำนวนปุ่ม : 16 ปุ่ม (4 แถว × 4 คอลัมน์)
แรงดันไฟฟ้าใช้งาน : 3.3V ถึง 5V
การใช้กระแสไฟ : โดยทั่วไป < 10mA
ประเภทอินเทอร์เฟซ : เมทริกซ์ (ต้องใช้พิน GPIO 8 พินสำหรับการเชื่อมต่อโดยตรง)
ประเภทปุ่ม : ปุ่มกดชั่วขณะ
ขนาด : แตกต่างกันไปตามรุ่น โดยทั่วไปประมาณ 70 มม. × 70 มม.
ประเภทขั้วต่อ : หัวต่อ 8 พิน หรือแผ่นบัดกรี
